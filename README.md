# S.A.M 2.0

 SAM stands for "Smoke And Mirrors"
 
 SAM uses Ollama AI to converse with users, as well as has Discord bot commands for controlling her memories and toggling certain actions

 ### Usage
 SAM will speak to you if you direct your message at the bot by a reply or by direct name dropping
 
 Use "$s help" for a list of commands
